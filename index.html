Навички

Що я використовую в роботі
HTML5
Саме він створює каркас сайту чи програми, а п'ята версія дозволяє створювати більш SEO-оптимізовану структуру одержуваного продукту.
CSS3
CSS є основною технологією всесвітньої павутини, поряд з HTML та JavaScript. Ця мова стилів дозволяє створювати абсолютно будь-який зовнішній вигляд сайту або програми.
JavaScript
Ця мова програмування дозволяє оживити все, що завгодно: слайдери, вікна, підказки, вкладки, отримання даних від сервера та багато іншого.
Система контролю версій Git
Git - потужна та складна розподілена система контролю версій, яка записує зміни у файл або набір файлів протягом часу і дозволяє повернутися пізніше до певної версії.
Технологія Flexbox
Технологія Flexbox дозволяє контролювати розмір, порядок та вирівнювання елементів по кількох осях, розподіл вільного місця між елементами та багато іншого.
CSS Grid
CSS Grid Layout представляє двовимірну сітку для CSS. Її можна використовувати для розміщення основних областей сторінки або невеликих елементів користувача інтерфейсу.

Figma
